# DACH Residency Progress Tracker

Welcome! This repository is designed to help residents track their progress across key software engineering skills and technologies.

## Getting Started

Follow these steps to set up your local environment and initialize the repository.

### 1. Install Python and Requirements

- Make sure you have Python 3.10 or higher installed
- Install Requirements: 
``` console
pip install -r requirements.txt
```

### 2. Run the start script
``` console
python scripts/start.py
```

### 3. Update your progress
Update your progress through the buttons near the appropriate topic and save any
resources to the corresponding folder!
